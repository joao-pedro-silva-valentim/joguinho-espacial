🚀👨‍🚀 Missão Resgate Espacial

Um jogo simples desenvolvido em linguagem C utilizando estruturas de repetição, condicionais e eventos aleatórios. O jogador assume o controle de uma nave espacial e precisa sobreviver o maior tempo possível enquanto realiza missões de resgate pelo espaço. 🌌

🎯 Objetivo do Jogo

O objetivo é sobreviver às ameaças espaciais enquanto acumula pontos realizando resgates de sobreviventes perdidos pela galáxia. Durante a missão, diversos eventos aleatórios podem acontecer, exigindo atenção e estratégia do jogador.

🧠 Conceitos Aplicados

✅ Estrutura sequencial
✅ Estruturas condicionais (if, else if, else)
✅ Estrutura de repetição (while)
✅ Geração de números aleatórios (rand)
✅ Entrada e saída de dados
✅ Interação com o usuário no terminal

⚠️ O projeto foi desenvolvido sem o uso de:

vetores;
matrizes;
funções;
procedimentos.
🎮 Como Funciona

O jogador inicia a partida com:

🟢 100 pontos de energia
⭐ 0 pontos de resgate

A cada rodada, um evento aleatório acontece:

Evento	Consequência
⛽ Encontrou combustível espacial	+20 de energia
👾 Ataque alienígena	-30 de energia
🌪️ Tempestade espacial	-15 de energia
🧍 Sobrevivente resgatado	+50 pontos

Após cada rodada, o jogador decide se deseja:

continuar explorando 🚀
ou encerrar a missão 🛑

O jogo termina quando:

a energia chega a zero;
ou o jogador decide sair.
